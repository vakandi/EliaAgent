## Analysis Workflow

### Step 1: Ingest Sources
Read all three system prompts and MEMORY.md

### Step 2: Mine Raw Data
Run grep commands above to find evidence

### Step 3: Create Wiki Pages
Document findings in brain/pages/:
```bash
# Create issue page
cat > /Users/vakandi/EliaAI/brain/pages/issues/issue-[date]-[name].md << 'EOF'
---
title: Issue Title
type: issue
date: YYYY-MM-DD
severity: high|medium|low
status: open
---

# Issue Title

## What Happened
Description.

## Evidence
Links to raw sessions.

## Root Cause
Why it happened.

## Resolution
How to fix.
EOF

# Create mistake page
cat > /Users/vakandi/EliaAI/brain/pages/mistakes/mistake-[date]-[name].md << 'EOF'
---
title: Mistake Title
type: mistake
date: YYYY-MM-DD
severity: high
recurring: true|false
---

# Mistake Title

## What Happened

## Impact

## Prevention
EOF
```

### Step 4: Update Index
Edit `/Users/vakandi/EliaAI/brain/index.md` to add new pages

### Step 5: Log Entry
```bash
echo "## [YYYY-MM-DD] analysis | System Review" >> /Users/vakandi/EliaAI/brain/log.md
```

### Step 6: Generate Report
Save full analysis to:
`/Users/vakandi/EliaAI/docs/elia-system-analysis-YYYY-MM-DD.md`
